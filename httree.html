Key Bindings

r = Add root level node
R = Rename current Folder
j = Select the next tree node
k = Select the previous tree node
, = Select the next sibling level tree node
. = Select the previous sibling level tree node
l = Expand the current tree node
h = Collapse the current tree node
u = go up a tree node level
C = Collapse all the tree nodes
E = Expand all the tree nodes
A = Add child tree node
a = Add next sibling tree node
] = Expand the treeview panel
[ = Shrink the treeview panel
g = Select top tree node
G = Select bottom tree node

y = Copy note node
p = Paste copied note node
                    
H = Move current node to the parent node
L = Make current node a child node
J = Move current node down
K = Move current node up
D = Delete the current node
i = Edit current note
/ = Search (begin with : to search labels)
f = Find/Replace text in the current note 
# = Show note links dialog
< = Show the history dialog
X = Show broken note links

s = Save the current file
S = Save as
T = Export current note as .txt
U = Upgrade (import existing data)

m = Bookmark current node
` = Show bookmarks

V = Author Vim keys help
! = Display options dialog

Key Bindings

i = Enter insert mode, where you can write text normally
a = Enter insert mode, where you can write text normally
h = Move left
l = Move right
j = Move down
k = Move up
0 = Jump to the beginning of the line
$ = Jump to the end of the line
w = Move forward one word
W or b = Move back a word
V = Enter line select mode
v = Enter standard select mode
y = Copy the selected text
x = Delete the character under the cursor
    (if in Select mode, cut the selected
    text to the clipboard)
d = Delete selected text
D = Delete everything right of the cursor
dd = Delete current row
gg = Go to the start of the note
#g = Go to the specified line (where # is a number)
G = Go to the end of the note


Note: If you enter a number first, some commands
      will repeat that many times.
